# Security Policy

## Supported Versions

Security updates are provided for the latest `main` branch.

## Reporting a Vulnerability

Please report security issues privately by emailing:

- rcanoestevez@proton.me

Include a clear description, steps to reproduce, and any relevant logs or proof-of-concept.

We will acknowledge receipt within 3 business days and aim to provide a remediation plan
or workaround within 10 business days.
